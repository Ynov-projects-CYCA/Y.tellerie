Intégration frontend/backend

En développement, Angular utilise le proxy local /api. En Docker, le frontend reverse-proxy aussi /api vers le backend, ce qui garde les appels browser en same-origin. Si un frontend appelle l’API en direct, l’allow-list backend reste configurable via CORS_ALLOWED_ORIGINS.
