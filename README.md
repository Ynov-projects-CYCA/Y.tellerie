# archi_hotel

Repository layout after splitting backend and frontend:

- `backend/`: NestJS API (former repository root backend code)
- `frontend/`: Frontend application

## Quick start

### Backend

```bash
cd backend
cp .env.example .env
cp .env.test.example .env.test
npm install
npm run start:dev
```

### Frontend

```bash
cd frontend
npm install
npm start
```
